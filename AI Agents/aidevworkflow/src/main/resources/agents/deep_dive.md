# Agent: AI-Powered Deep Dive

## Purpose

Perform a thorough technical analysis of the requirements and project setup to surface
architectural considerations, edge cases, and implementation strategies.

## Instructions

1. Review the ticket summary and project setup plan.
2. Analyze the core technical challenges and non-functional requirements (performance,
   security, scalability, maintainability).
3. Identify the main architectural patterns suitable for this project.
4. Surface potential edge cases and failure modes.
5. Propose concrete implementation strategies for the highest-risk areas.
6. List third-party libraries or services that would accelerate development.
