Code Diagram

Java Class Diagram
classDiagram
    class RAGPipeline {
        -OllamaClient ollamaClient
        -PromptBuilder promptBuilder
        -KnowledgeBaseSearcher searcher
        +generateFeedback(query, findings, code) String
    }
    class OllamaClient {
        -ChatLanguageModel model
        -String modelName
        +generate(prompt) String
        +isAvailable() bool
    }
    class PromptBuilder {
        +buildPrompt(query, findings, entries, code) String
        +buildSimplePrompt(query, code) String
    }
    class KnowledgeBaseSearcher {
        -String indexDirPath
        +search(queryStr, maxResults) List
    }
    class KnowledgeBaseIndexer {
        +indexKnowledgeBase(kbDir, indexDir) void
    }
    class CheckstyleAnalyzer {
        +analyze(file, config) List
    }
    class PMDAnalyzer {
        +analyze(file, ruleset) List
    }
    class AnalysisFinding {
        +String issue
        +String details
    }
    class KnowledgeEntry {
        +String title
        +String type
        +String description
        +List tags
    }
    class WebRunner {
        +main(args) void
    }
    class RagWebServer {
        -int port
        +start() void
    }

    RAGPipeline --> OllamaClient : uses
    RAGPipeline --> PromptBuilder : uses
    RAGPipeline --> KnowledgeBaseSearcher : queries
    KnowledgeBaseSearcher ..> KnowledgeEntry : returns
    CheckstyleAnalyzer ..> AnalysisFinding : produces
    PMDAnalyzer ..> AnalysisFinding : produces
    WebRunner --> RagWebServer : starts
    RagWebServer --> RAGPipeline : dispatches to
    RagWebServer --> OllamaClient : uses directly
    RagWebServer --> PromptBuilder : uses directly
