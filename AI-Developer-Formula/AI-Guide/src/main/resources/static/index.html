📖 What is MaskMe?

MaskMe is a lightweight Java library that masks sensitive fields using annotations. Instead of writing repetitive masking logic across your codebase, you annotate fields once and let the library handle the rest — across any framework.

❌ Without MaskMe (Hardcoded)

public UserDto maskUser(UserDto user) { return new UserDto( 1000L, "****", user.name() + "@masked.com", "****", "***-**-****", maskAddress(user.address()), LocalDate.of(1800, 1, 1), user.genderId(), user.genderName(), BigDecimal.ZERO, Instant.parse("1900-01-01T00:00:00Z") ); // Repeat for EVERY DTO... }

✅ With MaskMe (Annotation-Driven)

public record UserDto( @MaskMe(conditions = {AlwaysMask.class}, maskValue = "1000") Long id, @MaskMe(conditions = {MaskMeOnInput.class}, maskValue = "{email}-{genderId}") String name, @MaskMe(conditions = {AlwaysMask.class}) String password // ... annotate once, done! ) {} // Usage: MaskMeInitializer.mask(dto);

🚀 Key Features

  • Framework-agnostic — works with Spring, Quarkus, or pure Java
  • Annotation-driven — @MaskMe on fields, done
  • Conditional masking — mask based on runtime input, roles, environment
  • Field references — {fieldName} patterns in mask values
  • Nested object support — recursive masking through object graphs
  • Custom converters — override masking for any type
  • Scoped converter registry — global, thread-local, request-scoped
  • Thread-safe — ThreadLocal-based condition inputs
  • Multiple conditions per field — OR logic evaluation
  • Zero framework dependencies in core

🏛️ How It Works

1. Annotate → Mark fields with @MaskMe(conditions, maskValue)
2. Configure → Register framework provider & custom converters at startup
3. Mask → Call MaskMeInitializer.mask(dto, Condition.class, input)
4. Process → Library discovers fields → evaluates conditions → converts types → returns masked copy

🎯 Built-in Conditions

Condition Behavior Input Required
AlwaysMaskMeCondition Always masks the field regardless of input None
MaskMeOnInput Masks only when input equals "maskMe" String
Custom Conditions Implement MaskMeCondition for any business logic Any Object
